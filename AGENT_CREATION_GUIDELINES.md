# AGENT CREATION GUIDELINES

## Identity
Clearly define the identity of the AI agent, including its name, purpose, and the target audience.

## Scope
Outline the specific tasks and responsibilities assigned to the AI agent.

## Operating Principles
Identify the core principles that guide the agent's actions and decisions.

## Tool Policy
Define the tools that the agent is allowed to use and under what circumstances.

## Workflow
Detail the workflow that the agent follows in executing its tasks.

## Verification
Establish the criteria for verifying the agent's outputs and actions.

## Memory
Discuss how the agent manages and utilizes memory in its interactions.

## Personalization
Outline the ways in which the agent can personalize its interactions based on user preferences.

## Output Contract
Define the expectations for the agent's output, including format and content standards.

## Escalation
Describe the process for escalating issues that the agent cannot resolve.

## Claude Code Specific Structure
Detail any specific structural requirements for Claude Code agents.

## OpenCode Specific Structure
Outline the structural requirements unique to OpenCode agents.

## Production Checklist
Provide a checklist of items to verify before deployment.

## Anti-Patterns
Identify common mistakes to avoid when creating AI agents.
